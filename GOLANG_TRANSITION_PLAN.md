## 2. Выполненные улучшения

### 2.1 Исправление терминала (Hero.tsx)

**Проблема:** Терминал перекрывал текст заголовка "Go-разработка. С нуля до продакшена"

**Решение:**
- Изменена сетка с `lg:grid-cols-2` на `lg:grid-cols-[1fr,500px]`
- Уменьшена ширина терминала с 520px до 480px
- Добавлен `overflow-visible` для корректного отображения

```tsx
// Было
<div className="grid lg:grid-cols-2 gap-12 lg:gap-16 items-center">

// Стало  
<div className="grid lg:grid-cols-[1fr,500px] xl:grid-cols-[1fr,580px] gap-12 lg:gap-8 items-center">
```

### 2.2 Яркость цифр 01/02/03 (Program.tsx)

**Проблема:** Номера модулей были слишком тусклыми (`text-primary/30`)

**Решение:**
- Изменен цвет с `text-primary/30` на `text-primary`
- Теперь цифры яркие и хорошо читаемые на темном фоне

```tsx
// Было
<span className="font-serif text-3xl lg:text-4xl font-bold text-primary/30">

// Стало
<span className="font-serif text-3xl lg:text-4xl font-bold text-primary">
```

### 2.3 Текст "6-9 месяцев" (Program.tsx, Testimonials.tsx)

**Проблема:** "6-9 месяцев до оффера" звучит как обещание трудоустройства

**Решение:**
- Изменено на "6-9 месяцев продуктивной работы"
- Более честное и позитивное позиционирование

```tsx
// Было
<p className="text-sm text-muted-foreground mt-1">месяцев до оффера</p>

// Стало
<p className="text-sm text-muted-foreground mt-1">месяцев продуктивной работы</p>
```

## 3. Архитектурные обновления

### 3.1 Централизация контента

**Создан файл:** `src/config/content.ts`

Теперь весь текстовый контент централизован в одном месте для:
- Удобства обновления текстов
- Консистентности данных
- Возможности локализации в будущем

```typescript
export const SITE_CONTENT = {
  brand: {
    name: "ZaharGo",
    tagline: "Go-разработка. С нуля до продакшена.",
  },
  stats: {
    projects: 5,
    tasks: "150+",
    graduates: "50+",
    employmentRate: "87%",
    timeframe: "6-9",
    timeframeLabel: "месяцев продуктивной работы",
  },
  // ... остальной контент
};
```

### 3.2 Структура конфигов

```
src/config/
├── links.ts      # Ссылки (Telegram, etc.)
└── content.ts    # Весь текстовый контент
```
